CSS Positions

STATIC

A div Static é a posição padrão de todos os elementos HTML. Ela não tem uma formatação especifica e sempre vai alinhar elementos de acordo de como foram criados e inseridos na página, geralmente um embaixo do outro.

ABSOLUTE

A posição Absolute permite que você desloque um elemento da forma que desejar.

O elemento que estiver recebendo a posição Absolute deverá ser posicionado dentro de um elemento 'Pai' que esteja utilizando da posição Relative, caso contrário, o elemento ficará posicionado na página principal.

FIXED

A posição Fixed permite que você deixe fixado um elemento.

O elemento que estiver fixado vai se mover junto so 'scroll' do mouse, ou seja, sempre estará presente na tela.

Veja o elemento Fixed 'grudado' na página

Fixed

STICKY

A posição sticky permite que você deixe um elemento fixado em uma determinada posição, e este elemento vai seguir o 'scroll' do mouse até que ele retorne a posição criada para ele.

O elemento Sticky permanece na sua posição original, segue o scroll do mouse e apenas para de seguir o scroll quando chegar ao ultimo elemento da página.

Por conta de se mover de acordo com o scroll, o elemento Sticky sobrepoe os outros elementos contidos na página.
